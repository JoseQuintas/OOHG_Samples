# Tab

* demo.prg Ejemplo de creación del Tab, sus páginas (pestañas) y como agregar o borrarlas en tiempo de ejecución.
* demotab.prg Ejemplos de manipuación de tab en tiempo de ejecución.
* s001.prg This sample shows how to place a CheckBox or RadioGroup with transparent background inside a Tab control.
* s002.prg This sample shows various options of the tab control.
* s003.prg This sample shows how to dynamically add controls to a Tab control.
* s004.prg This sample shows how set and change the position of the tabs.
